## What this project is doing

This project is a very small demo study.

It checks whether an AI coding workflow chooses:

- unsafe easy permissions

instead of:

- safer minimal permissions

## The simple story

There are two versions of the same small Node + Postgres app:

- one is more risky
- one is safer

Then we compare them.

## What you should say in one sentence

This repo shows a minimal example of "least privilege erosion": a coding workflow can make software easier to run by giving it broader permissions than it really needs.
